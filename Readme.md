## Начало работы

Все очень просто. Нужно взять компьютер. Открыть любой браузер (желательно Chrome), зайти на сайт github.com, ввести свои данные и зарегистрироваться.

### Prerequisites

На компьютер нужно установить браузер Chrome и Git.
